Frontend Environment (/frontend)

2. Vite (Latest) + Build tool and development server

Backend Environment (/backend)
Target Deployment: Render

1. Express + Web framework for Node.js
2. TypeScript + Static typing for backend logic
3. SupabaseJS + Database and authentication client
4. Zod + Schema declaration and environment variable validation
5. Helmet + Security middleware for HTTP headers
6. CORS + Cross-Origin Resource Sharing policy
7. ts-node-dev + Development server with hot-reloading
8. dotenv + Loading environment variables
